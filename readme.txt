This plugin displays the post list using shortcode. The post list is loaded with ajax.

== Description ==

Post List Generatorは、記事（投稿、固定ページ）の一覧を表示するプラグインです。[このページ](http://residentbird.main.jp/bizplugin/news/)のような記事一覧ページを作成できます。
記事一覧の日付、タイトルは自動的に表示されます。また、タイトルをクリックすると該当のページが開きます。

= 特徴 =
* 記事一覧に表示するコンテンツに投稿、固定ページ、投稿＋固定ページを設定できます
* 表示順序を公開日順・更新日順・タイトル順から選択できます
* Ajaxを使って記事一覧を表示することで、一覧性と高速な表示を両立しています
* 一度に表示する記事の件数を15件、30件、50件から選択できます
* カテゴリを指定して記事の新着情報を表示させることができます

詳細な使用方法はこちらです: http://residentbird.main.jp/bizplugin/plugins/post-list-generator/

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
